\section*{Dataset}

Since no pre-existing dataset exists for this domain, we construct our own knowledge base from publicly available institutional sources.

\subsection*{Data Sources}

We collect data from three main sources: the \textbf{UL FRI website} (\href{https://www.fri.uni-lj.si}{fri.uni-lj.si}), covering study program descriptions, enrollment procedures, exam rules, and the academic calendar; the \textbf{UL central website} (\href{https://www.uni-lj.si}{uni-lj.si}), containing university-wide student regulations and administrative guidelines; and \textbf{official PDF documents}, such as the Student Statute and enrollment rulebooks.

\subsection*{Data Collection and Processing}

Web pages will be scraped and cleaned to retain only the main textual content, discarding navigational elements and footers. Each document will be stored alongside metadata including its source URL and retrieval date. PDF documents will be processed using text extraction libraries to produce clean plain-text representations. The collected documents will then be split into overlapping chunks to enable fine-grained retrieval of relevant passages, following standard RAG preprocessing practices.
