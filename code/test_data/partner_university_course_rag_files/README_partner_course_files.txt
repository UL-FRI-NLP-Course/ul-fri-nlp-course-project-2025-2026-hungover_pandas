PARTNER UNIVERSITY COURSE RAG FILES

This folder contains concise RAG-ready knowledge files for selected UL FRI Erasmus+ partner universities.

Included institutions:
- TU Graz
- TU Berlin
- NTNU
- University of Padova
- Vrije Universiteit Brussel
- University of Tartu
- Charles University MFF
- Reykjavík University

Purpose:
These files are designed to help an Erasmus RAG assistant answer questions about:
- where to find course catalogues,
- whether a university is relevant for Computer Science,
- which topic areas may be available,
- how exchange students should select courses,
- what limitations apply,
- and when students must consult the official catalogue or Learning Agreement coordinator.

Important:
The files do not replace official course catalogues. Course availability changes by semester.
The assistant should avoid guaranteeing course availability unless the current official catalogue confirms it.
